### Online Services

* [Hopain Tools](https://osint.hopain.cyou/Search.html), [Inteltechniques](https://inteltechniques.com/tools/Search.html), [IntelligenceX](https://intelx.io/tools?tab=general), [Aware OSINT Tools](https://www.aware-online.com/osint-tools/) ‚Äî These serve as starting points for numerous general and specialized search engines, all heavily oriented towards OSINT.
* [Shodan](https://www.shodan.io/) ‚Äî Search by ip, name, or domain
* [Fagan Finder](https://www.faganfinder.com/) ‚Äî Offers a panel featuring dozens of search engines catering to libraries, archives, and databases.
* [Dogpile](https://www.dogpile.com/), [iZito](https://www.izito.com/), [zapmeta](https://www.zapmeta.fi/) ‚Äî Metasearch engines that aggregate results from popular search engines like Google, Yandex, and Bing, presenting them on a single page.
* [Metaosint](https://metaosint.github.io/) ‚Äî A search engine for search engines, providing a user-friendly interface to access other tools.
* [Synapsint](https://synapsint.com/) ‚Äî A metasearch engine with the capability to search by IP, SSL, ASN, CVE, email, and phone numbers.
* [ThatsThem](https://thatsthem.com/people-search) ‚Äî A combined search engine for names, addresses, phone numbers, email addresses, and IP addresses.
* [Carrot2](http://project.carrot2.org/) ‚Äî A search engine equipped with a built-in text clustering algorithm, automatically grouping thematically related sources.
* [Isearchfrom](http://isearchfrom.com/) ‚Äî Simulates search queries to Google from different countries, aiding in understanding how regional restrictions affect search results.
* [Answerthepublic](https://answerthepublic.com/) ‚Äî An English-language search engine that provides common search queries for a given phrase.
* [IntelTechniques Tools](https://inteltechniques.com/tools/index.html) ‚Äî Some tools of Michael Bazzell on his website
* [IntelX Tools](https://intelx.io/tools) ‚Äî IntelX OSINT tools
* [OSINT Essentials](https://www.osintessentials.com/) ‚Äî Some of OSINT investigation tools
* [InfoSecurity OSINT Tools](https://infosecurity.by/osint-tools) ‚Äî Same as above
* [Doxbin](https://doxbin.com/) ‚Äî Doxxing pastes on people
* [OSINT Framework](https://osintframework.com/) ‚Äî OSINT framework tree
* [Rawsec's CyberSecurity Inventory : Reconnaissance](https://inventory.raw.pm/tools.html#titl...nnaissance) ‚Äî OSINT and Reconnaissance

### Utilities

* [S](https://github.com/zquestz/s) ‚Äî Allows searching 106 sources directly from the command line.
* [SpiderFoot](https://github.com/smicallef/spiderfoot) ‚Äî Automates search queries and exports results to CSV, JSON, or GEXF. Tailored for red team tasks, equipped with a built-in web interface.
* [Query-Server](https://github.com/fossasia/query-server) ‚Äî Sends search queries to multiple search engines and writes the results in CSV, JSON, or XML.
* [Recon-ng](https://github.com/lanmaster53/recon-ng) ‚Äî An open-source intelligence framework resembling the Metasploit Framework. Facilitates searches for logins, phone numbers, addresses, and publicly available financial statements.
* [Querytool](https://github.com/oryon-osint/querytool) ‚Äî An OSINT tool based on Google Sheets, designed for complex searches of people, email addresses, files, and more.
* [Maltego Community Edition](https://www.maltego.com/product-features/) ‚Äî A free OSINT tool for collecting information from various sources and identifying correlations between entities like people, email addresses, companies, websites, and domains.
* [sn0int](https://github.com/kpcyrd/sn0int) ‚Äî A framework for collecting and semi-automatically processing information about subdomains, IP addresses, compromised accounts, phone numbers, and social network profiles.
* [theHarvester](https://github.com/laramies/theHarvester) ‚Äî A console utility for collecting information for red teaming, conducting active and passive reconnaissance using various search engines. Gathers names, email addresses, IP addresses, subdomains, and URLs.
* [ReconSpider](https://github.com/bhavsec/reconspider) ‚Äî A crawler for searching IP addresses, emails, websites, organizations, and retrieving information from diverse sources.
* [Mr.Holmes](https://github.com/Lucksi/Mr.Holmes) ‚Äî Gathers information about user domains, names, and phone numbers using public sources and Google Dorks.
* [Datasploit](https://github.com/DataSploit/datasploit) ‚Äî An OSINT tool found in Kali or BlackArch Linux, designed to collect data by specific domain, email, username, or phone number, and save reports in text files, HTML, and JSON.
* [YaCy](https://yacy.net/) ‚Äî An open-source, decentralized search engine that enables users to deploy their own search engine.
