### Base64

```bash
# encode
echo "Script Kiddie" | base64 
# output
U2NyaXB0IEtpZGRpZQo=

# decode
echo "U2NyaXB0IEtpZGRpZQo=" | base64 --decode 
# output
Script Kiddie
```
